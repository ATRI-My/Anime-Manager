# 动漫管理工具安装说明

## 📦 生成的安装包

### 1. 完整安装包 (`dist/Anime-Manager-Setup-Final.exe`)
- **大小**: 136MB
- **类型**: 7-Zip自解压安装程序
- **功能**:
  - 图形化安装界面
  - 自动解压到 `%ProgramFiles%\Anime Manager`
  - 创建开始菜单快捷方式
  - 创建桌面快捷方式
  - 包含卸载脚本

### 2. 简单安装包 (`dist/Anime Manager Setup.exe`)
- **大小**: 136MB
- **类型**: 基础自解压程序
- **功能**: 解压后直接运行

### 3. 便携版本 (`dist/win-unpacked/`)
- **大小**: 约200MB（整个目录）
- **类型**: 绿色版，无需安装
- **主程序**: `Anime Manager.exe`

## 🚀 安装方法

### 方法1：完整安装（推荐）
1. 双击运行 `Anime-Manager-Setup-Final.exe`
2. 按照安装向导操作
3. 安装完成后会自动启动程序

### 方法2：便携版使用
1. 进入 `dist/win-unpacked/` 目录
2. 双击 `Anime Manager.exe`
3. 无需安装，直接使用

### 方法3：简单安装
1. 双击 `Anime Manager Setup.exe`
2. 选择解压目录
3. 进入解压后的目录运行程序

## 🔧 系统要求

- **操作系统**: Windows 10/11 (64位)
- **内存**: 至少4GB RAM
- **磁盘空间**: 至少500MB可用空间
- **屏幕分辨率**: 1024x768或更高

## 🗑️ 卸载方法

### 完整安装版卸载：
1. 进入 `%ProgramFiles%\Anime Manager`
2. 运行 `uninstall.bat`
3. 或手动删除安装目录和快捷方式

### 便携版卸载：
直接删除整个 `win-unpacked` 目录

## 🆘 常见问题

**Q: 安装程序无法运行？**
A: 请右键选择"以管理员身份运行"

**Q: 程序启动很慢？**
A: 首次启动需要加载资源，后续启动会更快

**Q: 如何备份数据？**
A: 数据默认保存在用户目录的 `.anime-manager` 文件夹中
