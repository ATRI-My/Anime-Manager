# Anime Manager 未打包版本使用说明

## 概述
本文档介绍如何运行和使用 Anime Manager 的未打包版本。未打包版本是一个完整的应用程序目录，包含所有运行时依赖，无需安装即可运行。

## 系统要求
- **操作系统**: Windows 10 或更高版本
- **内存**: 至少 4GB RAM
- **磁盘空间**: 至少 500MB 可用空间
- **屏幕分辨率**: 1280x720 或更高

## 快速开始

### 1. 获取应用程序
未打包的应用程序位于：
```
dist/win-unpacked/
```

### 2. 运行应用程序
双击以下文件启动应用：
```
dist/win-unpacked/Anime Manager.exe
```

或者使用命令行：
```bash
cd dist/win-unpacked
"Anime Manager.exe"
```

### 3. 首次运行
首次启动时，应用程序可能会：
- 创建用户数据目录
- 初始化配置文件
- 建立必要的数据库

## 应用程序结构

### 主要文件
```
dist/win-unpacked/
├── Anime Manager.exe          # 主可执行文件
├── resources/                 # 应用资源
│   └── app/                  # 应用代码和依赖
├── locales/                   # 本地化文件
├── *.dll                      # 系统库文件
└── *.pak                      # Chromium 资源文件
```

### 用户数据
应用程序数据存储在用户目录：
```
%APPDATA%\Anime Manager\
```

包含：
- 配置文件
- 数据库文件
- 日志文件
- 缓存数据

## 故障排除

### 常见问题

#### 1. 应用程序无法启动
**症状**: 双击无反应或立即关闭
**解决方案**:
- 检查系统要求是否满足
- 确保有足够的磁盘空间
- 以管理员身份运行
- 检查防病毒软件是否阻止

#### 2. 界面显示异常
**症状**: 界面元素错位或空白
**解决方案**:
- 重启应用程序
- 清除缓存：删除 `%APPDATA%\Anime Manager\Cache`
- 更新显卡驱动程序

#### 3. 功能异常
**症状**: 某些功能无法使用
**解决方案**:
- 检查日志文件：`%APPDATA%\Anime Manager\logs\`
- 重置配置文件（备份后删除）
- 重新生成应用程序

### 日志文件
应用程序日志位于：
```
%APPDATA%\Anime Manager\logs\
```

查看日志有助于诊断问题。

## 高级使用

### 命令行参数
```
"Anime Manager.exe" [选项]

选项：
  --debug          启用调试模式
  --reset-config   重置配置文件
  --data-dir=<路径> 指定数据目录
  --help           显示帮助信息
```

### 调试模式
启用调试模式查看详细日志：
```bash
"Anime Manager.exe" --debug
```

### 自定义数据目录
使用自定义数据目录：
```bash
"Anime Manager.exe" --data-dir="D:\AnimeData"
```

## 更新和维护

### 更新应用程序
要更新应用程序，替换整个 `dist/win-unpacked/` 目录。

### 备份数据
定期备份用户数据目录：
```
%APPDATA%\Anime Manager\
```

### 清理缓存
清理缓存可以解决一些性能问题：
1. 关闭应用程序
2. 删除 `%APPDATA%\Anime Manager\Cache`
3. 重新启动
