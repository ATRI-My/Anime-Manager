# 剧集编辑功能测试指南

## 测试步骤

### 步骤1: 打开写入板块
1. 启动Anime Manager程序
2. 点击顶部导航栏的"写入"标签

### 步骤2: 选择番剧
1. 在左侧"番剧列表"中选择一个番剧（如"咒术回战 第二季"）
2. 右侧会显示该番剧的剧集表格

### 步骤3: 测试剧集编辑功能

#### 添加新剧集
1. 点击剧集表格下方的"添加新行"按钮
2. 在弹出的模态框中填写：
   - 剧集编号: 4
   - 剧集标题: "涩谷事变 激战"
   - 剧集链接: https://example.com/jujutsu-2/ep4
   - 观看状态: 未选中（默认）
   - 备注: 可选
3. 点击"添加剧集"按钮
4. 应该看到Toast提示"剧集 '涩谷事变 激战' 添加成功"
5. 剧集表格中应该显示新添加的剧集

#### 编辑现有剧集
1. 在剧集表格中找到要编辑的剧集
2. 点击该剧集右侧的"编辑"按钮
3. 在弹出的模态框中修改信息：
   - 修改标题或链接
   - 勾选"已观看"复选框
4. 点击"更新剧集"按钮
5. 应该看到Toast提示"剧集信息更新成功"
6. 剧集表格中应该显示更新后的信息

#### 删除剧集
1. 在剧集表格中找到要删除的剧集
2. 点击该剧集右侧的"删除"按钮
3. 在弹出的确认对话框中点击"确定"
4. 应该看到Toast提示"剧集删除成功"
5. 剧集表格中应该不再显示该剧集

### 步骤4: 测试批量删除
1. 勾选多个剧集前的复选框
2. 点击"批量删除"按钮
3. 在弹出的确认对话框中点击"确定"
4. 应该看到Toast提示"剧集删除成功"
5. 选中的剧集应该被删除

## 验证要点

### 数据验证
1. **剧集编号验证**: 必须大于0，不能重复
2. **标题验证**: 不能为空
3. **链接验证**: 不能为空
4. **数据持久化**: 关闭程序后重新打开，修改应该被保存

### UI验证
1. **模态框功能**: 打开、关闭正常
2. **表单验证**: 必填项验证正常
3. **Toast反馈**: 操作成功/失败有提示
4. **状态更新**: 表格实时更新

### 错误处理
1. **重复编号**: 尝试添加重复编号的剧集应该失败
2. **空数据**: 尝试提交空标题或空链接应该失败
3. **无效编号**: 尝试提交0或负数的编号应该失败

## 预期结果

1. 剧集编辑功能应该正常工作
2. 所有操作应该有视觉反馈（Toast）
3. 数据应该正确保存到文件
4. 表单验证应该防止无效数据

## 故障排除

如果功能不正常，请检查：

1. **控制台错误**: 按F12打开开发者工具，查看Console标签
2. **数据文件**: 检查`%APPDATA%\anime-manager\anime-data.json`是否可写
3. **组件加载**: 确认EpisodeForm和EpisodeModal组件正确导入
4. **状态更新**: 确认selectedAnime状态正确更新
