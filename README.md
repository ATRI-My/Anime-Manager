# Anime Manager - 动漫管理器

一个简单易用的动漫资源管理桌面应用，完全由AI辅助开发完成。

## 🚀 主要功能

### 📋 动漫管理
- **添加/编辑动漫**：管理动漫标题、描述、标签和观看方式
- **剧集管理**：添加、编辑、删除剧集，标记观看状态
- **智能搜索**：快速查找动漫和剧集

### 🎯 核心特性
- **本地数据存储**：数据保存在本地JSON文件中，安全可靠
- **现代化界面**：简洁直观的标签页设计（查询、写入、设置）
- **实时验证**：表单输入实时验证，减少错误
- **批量操作**：支持批量删除剧集等操作

### ⚙️ 实用工具
- **自定义工具配置**：配置外部播放器或下载工具
- **数据文件夹管理**：一键打开应用数据目录
- **环境检查**：内置系统环境检测工具

## 📦 快速使用

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建应用
```bash
# 构建项目
npm run build

# 生成可执行文件
npm run pack
```

### 直接运行
- 双击 `start-app.bat` 启动应用程序
- 运行 `test-run.bat` 检查系统环境

## 📁 项目文件

### 核心代码
```
src/
├── main/           # Electron主进程
├── renderer/       # React界面
└── shared/         # 共享代码和类型定义
```

### 实用脚本
- `start-app.bat` - 启动应用程序
- `start-build.bat` - 启动构建版本
- `test-run.bat` - 运行环境测试
- `cleanup.bat` - 清理项目文件
- `setup.bat` - 安装程序脚本

### 数据存储
- 位置：`%APPDATA%\anime-manager\`
- 格式：JSON文件（anime-data.json, settings.json）
- 备份：建议定期备份数据文件夹
