SearchBar 建议功能演示

展示增强的SearchBar组件，支持标题建议和高亮匹配文本

功能说明

  • • 防抖搜索: 输入时自动防抖处理，避免频繁搜索
  • • 标题建议: 输入时显示匹配的动漫标题建议
  • • 高亮匹配: 建议中高亮显示与搜索词匹配的部分
  • • 点击选择: 点击建议项自动填充搜索框
  • • 外部关闭: 点击下拉框外部自动关闭建议
  • • 清除功能: 一键清除搜索内容

React 组件接口:

interface SearchBarProps {
  placeholder?: string;
  onSearch: (query: string) => void;
  className?: string;
  debounceDelay?: number;
  suggestions?: string[];
  onSuggestionSelect?: (suggestion: string) => void;
}
